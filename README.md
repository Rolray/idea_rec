# GeekMind - 想法记录

一个极客简约风格的想法记录应用，支持文本和语音输入，时间轴和看板双视图展示。

## 功能特性

- 用户注册和登录（基于 Supabase Auth）
- 文本输入和语音输入
- 倒序时间轴展示
- 模块化看板展示（想法/待办/完成）
- 想法编辑和删除
- 响应式设计（PC + 移动端）
- PWA 支持（可安装到主屏幕）
- 离线访问

## 部署

- 域名: https://idearec.xr-semi.space
- 托管: Vercel
- 数据库: Supabase

## 技术栈

- 前端: 原生 JavaScript + CSS
- 后端: Supabase (PostgreSQL + Auth + RLS)
- 部署: Vercel
